User-defined Chromosomes

Although GAUL provides several, commonly used, chromosome types there will always be occassions where a problem-specific chromosome will be highly beneficial. Creating a new chromosome type is straight forward because the core routines in GAUL were designed to be chromosome datatype agnostic.
