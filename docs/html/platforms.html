GAUL Documentation: Platforms

Development Platform:

Nearly all of the GAUL development is undertaken on either x86-based or Alpha-based hardware running RedHat Linux 7.2. These computers are therefore guaranteed to be supported. Requests for alternative ports are likely to be taken very seriously, although they are subject to the relevant system being available.

Potentially Supported Platforms:

  • Linux and other (mostly) POSIX-compliant UNIX OSs on any sensible hardware. e.g. *BSD, IRIX, Solaris.
  • Non-toy OSs which are POSIX-compliant. e.g. BeOS, QNX.
