GAUL Documentation: Evolving Neural Networks

Very Brief Neural Network Primer

Artificial neural networks are a wide class of algorithms based on simple models of the brain. Their main forte is in their ability to classify data and to make predictions by interpolation. Certain neural network models are suited towards associative learning, or characterisation, or non-linear clustering in high-dimensional space.

There is a huge range of neural network models in use.

Example NN

The particular NN model in this example is a very simple multilayer perceptron with a sigmodial activation function. This is trained using a Lamarckian GA, where the adaptation step is traditional back-propagation with momentum. Certian aspects of this NN model are definitely less than optimal, but is designed for illustrative purposes. In this case the topology of the NN is fixed, whilst the weights and certian learning parameters are optimised using the GA with the weights additionally modified via Lamarckian adaptation. One aspect which makes this interesting is that the neural networks' learning parameters are evolved along with the weights which side-steps the additional problem of selecting a reasonable set of learning parameters.

The code is in examples/evolvenn.c, with the neural network functionality provided in util/nn_util.c and util/nn_util.h. This is a fairly straightforward example for HelGA. The majority of the code deals with the custom chromosome, consisting of a simple neural network datastructure, which is used.
